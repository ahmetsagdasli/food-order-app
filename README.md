# Food Order App

Modern bir yemek sipariş uygulaması.

## Özellikler

- Modern UI/UX tasarımı
- Responsive tasarım
- TypeScript desteği
- Modern state yönetimi (Zustand)
- Veri yönetimi (React Query)
- Modern doğrulama sistemi (Zod)
- Test desteği (Jest)

## Kurulum

### Backend Kurulumu

```bash
# Backend klasörüne git
cd backend

# Bağımlılıkları yükle
npm install

# TypeScript derle
npm run build

# Sunucuyu başlat
npm start
```

### Frontend Kurulumu

```bash
# Frontend klasörüne git
cd frontend

# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev
```

## Teknolojiler

### Frontend
- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Query
- Zustand
- React Router v6

### Backend
- Node.js
- Express 5.0
- TypeScript
- Zod
- Jest
